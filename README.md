# Power Transform Revisited: Numerically Stable, and Federated

## Installation

1. Create a Python environment:

```bash
conda create -n powertf python=3.11
conda activate powertf
```

2. Clone this repository:

```bash
git clone https://github.com/xuefeng-xu/powertf.git && cd powertf
```

3. Install dependencies:

```bash
pip install -e .
```

## Reproduction

For training experiments, see [train/README.md](train/README.md).

For numerical experiments, see [numerical/README.md](numerical/README.md).
